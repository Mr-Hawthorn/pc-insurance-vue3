# PC录单

*切换node版本至12.0.0及以上*
## 本地调试
`npm run dev`

## 本地打包
`npm run build` // 为生产环境构建产物

## 本地预览线上
`npm run serve` // 本地预览生产构建产物 build打包后才能看到页面
